Tusmo

A Somali programming language that compiles to C.

Quick Start

Linux/macOS

curl -fsSL https://raw.githubusercontent.com/Tusmolang/Tusmo/main/install.sh | bash
tusmo hello.tus

Windows (PowerShell)

irm https://raw.githubusercontent.com/Tusmolang/Tusmo/main/install.ps1 | iex
tusmo hello.tus
